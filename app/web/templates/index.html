About AICodeMentor

AICodeMentor is a runtime environment designed for building applications with integrated autonomous AI agents. It provides a simple REST API for file handling and workflow execution, plus a minimal browser UI for exploration and orchestration.

  • Execute Markdown-based workflows via HTTP.
  • Manage files and folders (upload, download, delete, create).
  • Authenticate once via a secure cookie; explore with Swagger UI.
